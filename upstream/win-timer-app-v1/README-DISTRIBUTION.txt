TaskTimer — таймер задач для Windows
=====================================

Запуск: дважды щёлкните TaskTimer.exe. Python устанавливать не нужно.
Данные (задачи и настройки) сохраняются автоматически в вашем профиле Windows.


Основное
--------
• Задачи привязаны к дням: создайте задачу, запустите таймер «Старт», приостановите «Стоп».
• Справа отображается текущая активная задача и накопленное время за сегодня по ней.
• Завершённые задачи можно снова открыть (возобновить) или скрыть фильтром «Только незавершённые».


Напоминание «продолжать?»
-------------------------
Через заданный интервал (по умолчанию 40 минут после старта или после ответа «Продолжить»)
приложение спрашивает, работаете ли вы ещё над задачей. Если не ответить вовремя, таймер остановится.
Интервал меняется в меню: Настройки → Параметры…


История времени
---------------
У каждой задачи кнопка «История»: просмотр интервалов, правка вручную, добавление записи за прошедшее время, удаление строки.


Режим концентрации
------------------
Вынесен на отдельную вкладку «Фокус» (основной экран остаётся только с задачами).
Отдельный обратный отсчёт (5–40 мин) — напоминание сфокусироваться без отвлечений, не привязан к задаче.


Трей и плавающий таймер
-----------------------
Окно можно свернуть; иконка в области уведомлений — открыть окно или выйти. Настройки доступны и из меню трея.
Когда окно свёрнуто в трей, поверх всех окон появляется маленький полупрозрачный виджет с текущей
задачей и затраченным временем; на нём есть кнопки старт/стоп. Виджет можно перетаскивать мышью,
двойной щелчок — развернуть окно обратно.


Технические заметки
-------------------
• Windows, 64‑бит. Первый запуск exe может быть чуть медленнее.
• Перенос на другой ПК: достаточно одного файла TaskTimer.exe, история задач на новом компьютере начнётся с нуля.
