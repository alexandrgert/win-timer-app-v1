Режим концентрации, доработки списка задач и панели таймера, исправления окна «Настройки» и layout на узких экранах.

**Режим концентрации**
- Пресеты 5 / 10 / 20 / 30 / 40 мин, обратный отсчёт, учёт времени в статистике
- Блок в правой колонке под «Завершить задачу», без перекрытия кнопок
- Плавающий виджет и трей показывают режим концентрации

**Список задач и таймер**
- Описание по клику, повторный клик сворачивает строку
- Кнопки действий в footer карточки; «Старт» по центру
- Панель таймера: только название и цифры, корректный перенос длинных названий

**Окно и настройки**
- Минимальные размеры окна — фильтры и элементы таймера не обрезаются
- «Настройки»: высота по содержимому, кнопки на всю ширину, прокрутка на маленьком экране
- WebDAV-статус после синхронизации отображается полностью
