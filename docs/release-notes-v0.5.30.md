## Что нового

### WebDAV — синхронизация между устройствами

- **Отдельные кнопки «Скачать и объединить» / «Загрузить сейчас»** — на десктопе (subbar) и в Android (главный экран); вместо одной неоднозначной кнопки облака.
- **Периодическая проверка сервера** — настраиваемый интервал (`sync_interval_minutes`); при изменениях с другого устройства запрос «Скачать и объединить?» / «Позже».
- **«Позже» с таймером** — повтор напоминания через 5 / 10 / 15 / 30 / 60 мин для той же версии на сервере; при новом hash на сервере — сразу новый запрос.
- **Pull-before-push** — перед загрузкой merge с сервером; meta-файл `.sync-meta.json` с SHA-256.
- **Проверка по hash файла** — periodic check скачивает `data.json` и сверяет hash (устаревший meta не скрывает реальные изменения).
- **Merge статуса задачи** — при слиянии учитываются `status` и `completed_at`: завершённая на одном устройстве не теряется; активная сессия → RUNNING.
- **Объединение сессий** — union по `session.id` с сохранением более «богатых» интервалов; fix парсинга Android-меток `+03:00` (корректное время на десктопе).
- **Android: полноценный WebDAV** — экран настроек, WorkManager в фоне, in-app монитор на переднем плане (без дубля «диалог + уведомление»).
- **Android: минимум 15 мин** для фоновой проверки (ограничение ОС); в открытом приложении — точный интервал. Подсказка в настройках.

### Список задач — десктоп

- **Даты «Создана» / «Завершена»** — read-only в шапке развёрнутой карточки (вкладка «Все»); формат `ДД.ММ.ГГГГ ЧЧ:ММ`.
- **Адаптивная шапка** — при нехватке места мета переносится над счётчиками «сег. · всего».
- **Исправлено наложение UI** — при раскрытой карточке на «Все» и переключении на «Сегодня» / «В работе» строки больше не перекрываются.
- **Стабильная пересборка списка** — без `processEvents()` при смене вкладки/даты.

### Android — задачи

- **Вкладка «Все»** — завершённые задачи зачёркнуты; строки «Создана» / «Завершена».
- **«Возобновить»** — для завершённых задач (как «Возобновить» на десктопе): сброс COMPLETED и старт новой сессии.
- **Фильтры** — Сегодня / В работе / Все.

### Настройки

- **WebDAV** — расширенная вкладка (интервал, напоминание «Позже», sync on startup/shutdown); увеличена минимальная высота окна настроек под WebDAV.

## Исправления и внутреннее

- **Normalize running** на Android — выбор «победителя» по instant, а не лексикографическому сравнению строк дат.
- **`sync_webdav_now`** — после pull загрузка без повторного download (`push_merged_state`).
- Удалён неиспользуемый API редактирования дат задачи (`parse_task_datetime`, параметры `created_at`/`completed_at` в `update_task`).
- **CI:** проверка совпадения `versionName` Android с `pyproject.toml` перед сборкой APK.
- **`.gitignore`:** keystore, `data.json`, `.gradle-user-home`.
- Расширены тесты: merge status, WebDAV periodic/stale meta, UI карточки, Kotlin DataMerge / TaskRepository.

## Как обновиться

### Linux

```bash
wget https://github.com/alexandrgert/timer-app/releases/download/v0.5.30/tasktimer-link-b24-0.5.30-amd64.deb
sudo dpkg -i tasktimer-link-b24-0.5.30-amd64.deb
sudo apt-get install -f
```

### Android

Скачайте `tasktimer-link-b24-0.5.30-android.apk` со [страницы релиза v0.5.30](https://github.com/alexandrgert/timer-app/releases/tag/v0.5.30) или установите локально:

```bash
adb install -r dist/tasktimer-link-b24-0.5.30-android.apk
```

Данные (`data.json`) и настройки WebDAV/Bitrix24 сохраняются при обновлении поверх текущей версии с той же подписью APK.
