# TaskTimer link B24 — версия 0.5.23

## Что нового

### Режим концентрации

- **Блок под таймером** — пресеты 5 / 10 / 20 / 30 / 40 минут, обратный отсчёт и кнопка «Остановить таймер».
- **Учёт времени** — при старте создаётся задача «Концентрация · … · ДД.ММ.ГГГГ ЧЧ:ММ»; время попадает в статистику и историю.
- **Пауза рабочей задачи** — при запуске фокуса текущая задача ставится на паузу; после окончания или ручной остановки предлагается возобновить.
- **Плавающий виджет и трей** — в мини-таймере и подсказке иконки отображается режим концентрации.
- **Всё в правой колонке** — отдельная вкладка «Фокус» убрана; блок расположен под «Завершить задачу» и не перекрывает кнопки.
- **Пресеты и остановка** — видны при минимальной высоте окна.

### Список задач

- **Описание по клику** — при выборе строки показывается описание с переносом длинных строк и авто-высотой.
- **Повторный клик сворачивает** строку — описание и панель действий скрываются.
- **Кнопки внизу карточки** — «Старт», «Изменить», «Завершить» и др. переносятся в footer, не уезжают за край при прокрутке.
- **Кнопка «Старт»** — выровнена по центру белой полосы действий.
- **Название задачи** — увеличена минимальная ширина в свёрнутой строке; длинные заголовки не обрезаются преждевременно.

### Панель таймера

- **Только название и цифры** — описание убрано из блока «Таймер» (остаётся в списке при клике по задаче).
- **Синие цифры** — фиксированная высота по метрикам шрифта; верх цифр больше не обрезается.
- **«СЕГОДНЯ» / «ВСЕГО»** — не наезжают на основной счётчик; карточка пересчитывает высоту при длинном названии.
- **Длинное название** — корректный перенос в карточке таймера.

### Окно и панель фильтров

- **Минимальная ширина окна** увеличена — кнопки «Сегодня», «В работе», «Все» и «＋ Новая задача» не обрезаются.
- **Минимальная высота окна** — нижние элементы таймера не скрываются при уменьшении окна.
- **«Сегодня всего: …»** — фиксированная ширина по образцу, без «прыжков» при смене даты.

### Окно «Настройки»

- **Минимальная высота по содержимому** — диалог подстраивается под обе вкладки (Битрикс24 и WebDAV); текст с переносом строк учитывается при сужении окна.
- **Кнопки на всю ширину** — «Проверить соединение», «Определить с портала» и кнопки WebDAV не обрезаются справа при минимальной ширине (~520 px).
- **Прокрутка на маленьком экране** — если контент не помещается по высоте, окно ограничивается размером экрана, лишнее прокручивается.
- **Поля формы** — стабильная минимальная высота полей и стрелки у spinbox на вкладке Битрикс24.
- **Статус WebDAV** — после синхронизации сохранённый статус (включая длинные ошибки) корректно отображается и пересчитывает высоту блока.

## Как обновиться

### Linux

```bash
wget https://github.com/alexandrgert/timer-app/releases/download/v0.5.23/tasktimer-link-b24-0.5.23-amd64.deb
sudo dpkg -i tasktimer-link-b24-0.5.23-amd64.deb
sudo apt-get install -f
```

Локальная сборка:

```bash
sudo dpkg -i tasktimer-link-b24-0.5.23-amd64.deb
sudo apt-get install -f
```

### Windows / macOS / Android

Скачайте нужный файл на [странице релиза v0.5.23](https://github.com/alexandrgert/timer-app/releases/tag/v0.5.23).

Данные (`data.json`) и настройки в `~/.config/tasktimer/` сохраняются при обновлении.
